Scenario

A software house wants an intelligent hiring system.

Inputs
Name
Age
CGPA
Skills Count
Experience
Expected Salary
Communication Score (1–10)
Logic
Reject Conditions

Reject if:

Age < 18
CGPA < 2.7
Skills < 3

Flag as suspicious if:

Expected Salary > 400000
AND Experience = 0
Hiring Levels
Intern
CGPA ≥ 2.7
Skills ≥ 3
Junior Developer
CGPA ≥ 3.0
Skills ≥ 5
Experience ≥ 1
Software Engineer

Skills ≥ 7
Communication ≥ 7
